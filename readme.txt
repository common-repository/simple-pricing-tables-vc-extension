== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Simple Pricing Tables - VC Extension' and install it
3. Open Visual Composer Page Builder
4. You can find Simple Pricing Table addon in "Content" tab of visual composer
